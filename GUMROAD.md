# IPIPQ - Gumroad 产品页面

## 产品标题
AI File Auto Organizer - 自动整理下载文件夹

## 产品描述

**你的下载文件夹是不是一团乱？**

每天下载的文件堆成山，找东西要翻10分钟？

IPIPQ 是一个 AI 文件自动整理工具，帮你：

✅ **自动识别文件类型** - 智能识别图片、文档、视频、代码等 50+ 种格式  
✅ **自动分类归档** - 按类型和日期自动整理到对应文件夹  
✅ **自动命名优化** - 清理乱码字符，处理重名文件

**使用方法超简单：**
```bash
python organize.py
```

完成！你的 Downloads 文件夹现在井井有条。

**整理效果：**
- 整理前：合同.pdf、IMG_1234.jpg、数据.xlsx、script.py...（一堆杂乱文件）
- 整理后：Documents/2024-01/、Images/2024-01/、Code/2024-01/...（井井有条）

**支持格式：**
- 图片：jpg, png, gif, psd, ai, sketch...
- 文档：pdf, doc, docx, txt, pages...
- 表格：xls, xlsx, csv, ppt...
- 视频：mp4, mov, avi, mkv...
- 音频：mp3, wav, aac, flac...
- 压缩包：zip, rar, 7z, dmg...
- 代码：py, js, html, java, go...
