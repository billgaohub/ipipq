使用方法

# 整理默认 Downloads 文件夹

$ python organize.py

# 整理指定目录

$ python organize.py /path/to/folder

# 预览模式（不实际移动）

$ python organize.py --dry-run

# 撤销上一次整理

$ python organize.py --undo
