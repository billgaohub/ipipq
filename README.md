# IPIPQ - AI File Auto Organizer

> 自动整理你的下载文件夹，让桌面永远干净

## ✨ 核心功能

- 🤖 **自动识别文件类型** - 智能识别图片、文档、视频、代码等 50+ 种格式
- 📁 **自动分类归档** - 按类型和日期自动整理到对应文件夹
- ✨ **自动命名优化** - 清理乱码字符，处理重名文件

## 🚀 快速开始

### 1. 下载

```bash
curl -O https://ipipq.com/ipipq-v1.0.0.zip
unzip ipipq-v1.0.0.zip
```

### 2. 运行

```bash
python organize.py
```

完成！你的 Downloads 文件夹现在井井有条。
